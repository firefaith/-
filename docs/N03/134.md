# 日本的武士道与国际的新伦理
(一九二一年十二月二日)
我们一度到日本、便联想及于日本的“武上道”、“大和魂”.“武士道”与”大和魂”、可以说是日本的国粹，日本民族的精神。口本民族所以能在世界有现在的地位，全靠这种的精神。
“武士道”、“大和魂”的精神，究竟是一种甚么精神呢？明的说，就是一种任侠好义的精神、扶弱摧强的精神，为公家牺牲个人的精神，日本人有这种精神，所以能爱国而为国死，爱恋人而常至于情死。H本的社会，所以常有壮烈慷慨的牺牲行为，刺激一般人的精神，使之感奋兴起。过去数十年的国家主义之所以成功，全是这种精神的产物。
但是自从前的日本人，只把这种精神限域于自国以内的社会伦理，没有把他发扬光大应用到国际的社会伦理上去。换句话说，就是从前的日本人，发扬这种精神于爱国的行为的地方多，发扬于爱人的行为的地方少，甚至把这种精神恶用了。我很相信在最近的将来，口本人一定能把那“武十道”、“大和魂”就是任侠好义、扶弱抑强的精神发扬光大，应用于国际的生活。使之不至再蹈恶用的覆辙。以实现一种人道主义的国际的新伦理。（一九二一年、十二、二，在北京)署名：北京大学教授李大钊
东方时论》第7卷第1号
